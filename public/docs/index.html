Introduction

Base URL: https://psrp.online

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Student Authentication

Register a new student.

This endpoint allows a new student to register by providing their details.

Example request:
curl --request POST \
    "https://psrp.online/api/student/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"first_name\": \"John\",
    \"last_name\": \"Doe\",
    \"email\": \"john.doe@example.com\",
    \"password\": \"secret123\",
    \"middle_name\": \"A\",
    \"password_confirmation\": \"secret123\"
}"

const url = new URL(
    "https://psrp.online/api/student/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "password": "secret123",
    "middle_name": "A",
    "password_confirmation": "secret123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "message": "Student registered successfully",
    "access_token": "2|a1s2d3f4g5h6j7k8l9m0",
    "token_type": "Bearer"
}

Example response (422):


{
    "errors": {
        "email": [
            "The email has already been taken."
        ]
    }
}

Request      

POST api/student/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

first_name   string   

The first name of the student. Example: John

last_name   string   

The last name of the student. Example: Doe

email   string   

The email address of the student. Must be unique. Example: john.doe@example.com

password   string   

The password for the student account (minimum 8 characters). Example: secret123

middle_name   string  optional  

The middle name of the student. Example: A

password_confirmation   string   

Password confirmation (must match the password). Example: secret123

Student login.

This endpoint allows a student to log in by providing their email and password.

Example request:
curl --request POST \
    "https://psrp.online/api/student/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"john.doe@example.com\",
    \"password\": \"secret123\"
}"

const url = new URL(
    "https://psrp.online/api/student/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "john.doe@example.com",
    "password": "secret123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "access_token": "2|a1s2d3f4g5h6j7k8l9m0",
    "token_type": "Bearer"
}

Example response (401):


{
    "message": "Invalid login details"
}
